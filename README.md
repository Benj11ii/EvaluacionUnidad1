# EvaluacionUnidad1
Cumplimiento obligatorio para unidad1 Frontend-IPSS
---
## Descripción

Página web corporativa para **Coningenio**, empresa ficticia de consultoría y desarrollo de software. Desarrollada como evaluación sumativa de la Unidad 1 del curso Desarrollo Frontend.

El sitio presenta los servicios de la empresa, información institucional y un formulario de contacto con validaciones del lado del cliente.

---

## Estructura del proyecto

```
EvaluacionUnidad1/
├── index.html        # Home — presentación y llamada a la acción
├── servicios.html    # Nuestros Servicios — 4 servicios detallados
├── nosotros.html     # Nosotros — misión, visión y valores
├── contacto.html     # Contáctenos — formulario con validaciones
├── style.css         # Estilos globales + modo día/noche + responsive
└── script.js         # Lógica JS: tema, validaciones, cambios dinámicos
```

---

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 (variables, Grid, Flexbox, media queries)
- JavaScript vanilla (ES6+)

---

## Funcionalidades implementadas

- **Modo día / noche** — alternancia con transición suave, preferencia guardada en `localStorage`
- **Diseño responsivo** — adaptado a iPhone 14 Pro Max, Samsung Galaxy S20 Ultra, iPad Air, iPad Pro y Desktop
- **Formulario de contacto** con validación de 3 campos y mensajes de error dinámicos
- **Navegación sticky** presente en todas las páginas
- **Tarjetas con hover** animado en servicios y nosotros
